## Anotações

### Inicializando Typescrpt

```bash
npm install typescript @types/node -D
npx tsc --init
```

Usando o repositório [tsconfig/bases](https://github.com/tsconfig/bases), buscar a versão do NodeJS correspondente ao projeto. Nesse caso, [tsconfig Node 22](https://github.com/tsconfig/bases/blob/main/bases/node22.json).
